# Full-Stack Todo Web Application

This project is a full-stack, multi-user Todo web application.

**Phase II is currently under development.**

This application is being built using a spec-driven, agent-led development process. The goal is to create a modern, production-style application with a Next.js frontend, a FastAPI backend, and a PostgreSQL database for persistent storage.

Please see the `specs` directory for detailed information about the features and architecture.
