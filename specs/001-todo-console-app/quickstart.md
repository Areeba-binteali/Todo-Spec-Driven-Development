# Quickstart: Full-Stack Todo Web Application

This guide explains how to set up and run the full-stack application, which consists of a Next.js frontend and a FastAPI backend.

**Note: This application is currently under development as part of Phase II.** Detailed setup and run instructions will be provided once the initial implementation is complete.

## Prerequisites

The final application will require:
- Node.js and npm/yarn for the Next.js frontend.
- Python for the FastAPI backend.
- A PostgreSQL database (e.g., Neon Serverless).
- Configuration for environment variables (API keys, database URLs, etc.).

Further instructions will be added as development progresses.
